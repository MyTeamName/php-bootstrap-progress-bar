# PHP Bootstrap Progress Bar

Create static [Bootstrap Progress Bars][1] using PHP.

## Installation

Install into your project using [composer][2].

    composer require jpuck/php-bootstrap-progress-bar

## Usage

See [example.php][3] for usage.
